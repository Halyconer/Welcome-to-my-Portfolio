# What is this?

Personal portfolio website featuring interactive demos and my resume.

## Tech Stack

**Frontend:** HTML, CSS, JavaScript, Google Fonts  
**Backend:** Python Flask, Raspberry Pi deployment  
**Smart Home:** Yeelight bulbs, nginx reverse proxy

## Quick Start

1. Clone the repository
2. Open `index.html` in a web browser
3. Try the interactive demos!

For backend setup, see [DEPLOYMENT.md](DEPLOYMENT.md)
